id

id는 페이지 내에서 특정 위치나 태그를 지정하는 것이기 때문에 오직 페이지 내에서 한 번만 사용할 수 있습니다. CSS에서 정의하는 방법은 '#id name'과 같이 적용합니다.

class

class는 자주 사용되는 스타일에 적용합니다. CSS에서 정의하는 방법은 '.class name'과 같이 적용합니다. 


<div id="wrap">
    <header id="header"> </header>
    <nav id="nav"> </nav>
    <section id="contents">
        <aside class="aside"> </aside>
        <section class="content">
                <article class="article1"> </article>
                <article class="article2"> </article>
                <article class="article3"> </article>
        </section>
    </section>
    <footer id="footer"> </footer>
</div>
