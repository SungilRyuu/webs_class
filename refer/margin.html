CSS margin

margin

margin은 요소의 네 방향 바깥 여백 영역을 설정합니다. margin을 어떠한 방식으로 적용할 수 있는 지 알아보겠습니다. 

margin-top: 10px;
margin-right: 11px;
margin-bottom: 12px;
margin-left: 13px;

가장 기본적인 margin 적용 방법입니다. 각 면을 직접 선언하여 적용합니다. 


margin: 10px;                               /* 위,오른쪽,아래,왼쪽 : 10px */
margin: 10px 11px;                      /* 위,아래:10px 왼쪽,오른쪽:11px */
margin: 10px 11px 12px;            /* 위:10px 왼쪽,오른쪽:11px 아래:12px */
margin: 10px 11px 12px 13px;  /* 위:10px 오른쪽:11px 아래:12px 왼쪽:13px */

margin을 이용하여 여러 면에 여백을 적용하는 방법입니다. 위쪽을 기준으로 시계방향으로 여백을 설정할 수 있습니다. 


margin: auto;                    /* 자동값, 기본값 */
margin: 0 auto;               /* 가운데 정렬(블록구조) */

속성 값을 auto로 설정하면, 수평 또는 수직의 중앙으로 자동 정렬해줍니다.

padding

padding은 요소의 네 방향 안쪽 여백 영역을 설정합니다. 

padding-top: 10px;
padding-right: 11px;
padding-bottom: 12px;
padding-left: 13px;

padding: 10px;                                    /* 위,오른쪽,아래,왼쪽 : 10px */
padding: 10px 11px;                           /* 위,아래:10px 왼쪽,오른쪽:11px */
padding: 10px 11px 12px;                 /* 위:10px 왼쪽,오른쪽:11px 아래:12px */
padding: 10px 11px 12px 13px;       /* 위:10px 오른쪽:11px 아래:12px 왼쪽:13px */

padding도 margin과 마찬가지로 위쪽을 기준으로 시계방향으로 여백을 설정할 수 있습니다.
참고로, padding의 경우 auto가 없습니다.
