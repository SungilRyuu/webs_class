2. 구구단 출력하기

2.1 for문을 이용하여 2단부터 9단까지 출력하시오!

{
    for(let i = 2; i <= 9; i++){
        document.write("<br>");
        
        for (let j = 1; j <= 9; j++){
            let sum = i * j;
            document.write(i + " * " + j + " = " + sum, "<br>");
        }
    }
}

2.2 while문을 이용하여 2단부터 9단까지 출력하시오!

{
    let i = 2;
    let j = 1;
            
    while (i <= 9) {
        while (j <= 9) {
            let sum = i * j;
            document.write(i + " * " + j + " = " + sum, "<br>");
            j++;
            }
        document.write("<br>");
        i++;
        j = 1;
    }
    
}

2.3 사용자가 원하는 단을 받아 출력하시오!

조건 : 사용자가 원하는 단은 prompt()를 사용합니다. 클릭

{
    let num, result; 

    document.querySelector(".btn03").addEventListener("click", () => {
        num = prompt("출력할 구구단 숫자를 입력해주세요!!", "2");

        for(let i = 1; i <= 9; i++){
            result = num + " * " + i + " = " + (num * i ) + "<br>"
            document.querySelector(".result03 > div").innerHTML += result;
        }
    });
}
